Sea Turtle Conservancy

This is a redisign for a non-profit

About the project
This was a UI redisign for a non-profit called sea turtle conservancy, whose mission is to help spread awareness and protect the lives of Sea Turtles. The goal of this project was to redisign the site in a more user friendly manner to invite more web traffic to the site. This project was done in a 4 member team, including my self my other members were Amy Vasquez, Armari manning, and Derek Hickman.
My Role:
  1. User researcher & Analysis
  2. Persona Creation
  3. Wireframes
  4. Mobile UI Desigin & Prototyping
  5. Usability Testing

Research & problem definition

Problem
The Sea Turtle Conservancy website is not user friendly, does not clearly convey its message. Neither does it clearly promote donating, adopting, or volunteering with the organization. Which is causing lower interest in working with and donating to the Sea Turtle Conservancy.
Research

We first conducted 4 user interviews, as well as created 2 user surveys, One survey for understanding usersâ€™ experience with nonprofit groups in general, and one to understand their experiences with conservancy groups.

Some of the key findings we found were:

  1. Most users have either donated, volunteered, or done both.
  2. Most users had not adopted an animal before.
  3. Users worked with their organizations because it made them feel good to give back, and benefited their community.
  4. Some users had trouble understanding what the purpose of the website was and felt that it took time to get through the information on the homepage.
Affinity Diagram
After analyzing all the data we retrieved from the interviews we made an affintiy diagram.
Competitor Analysis
We conducted a competitor analysis on 4 different sites and what made them succesful.
